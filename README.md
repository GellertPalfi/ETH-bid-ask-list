#### Lists okex and binance bid and ask prices for Etherum using Django and python requests modul.  
  
#### Steps to run project:  
 -  Clone repo  
 -  Go to project folder  
 -  Create a virtual enviroment(in this case the name is "venv")  
&emsp;with the "virtualenv venv" command  
 -  Activate the enviroment:  
&emsp;source venv/bin/activate  
 -  Install all required packages:  
&emsp;pip install -r requirements.tt  
 -  Initalieze db migrations:  
&emsp;python manage.py makemigrations  
 -  Execute db migrations:  
&emsp;python manage.py migrate  
 -  Run server  
&emsp;python manage.py runserver
